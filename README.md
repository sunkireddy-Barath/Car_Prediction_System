# üöó Car Price Prediction --- Machine Learning Project

This project builds a **Car Price Prediction Model** using machine
learning techniques.\
The goal is to analyze various car features (such as brand, model,
mileage, engine, fuel type, etc.) and predict the approximate market
value of a car.

This repository contains the Jupyter Notebook
**`Cars_price_prediction.ipynb`**, which includes the complete workflow
from data loading to model evaluation.
